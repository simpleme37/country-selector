# React 元件需求說明

此元件應該渲染一個國籍或國碼選擇器(見設計稿)，請自行設計所需的 props

## 排序邏輯

1. **預設選單**: 先顯示熱門國家，然後按國家縮寫（`shortName`）A-Z 分組排序。
2. **完全匹配**: 任一欄位（國碼、中文名、英文名、縮寫）完全相同時優先顯示，內部按國家縮寫 A-Z 排序。
3. **部分匹配**: 任一欄位包含搜尋詞時，按國家縮寫 A-Z 排序。
4. **多欄位匹配**: 支援國碼（`code`）、中文名（`zhName`）、英文名（`enName`）、縮寫（`shortName`）的完全匹配和部分匹配。

## 行為

- 按下 Enter 鍵時，選擇排序結果中的第一筆資料。
- blur 時也要選擇排序結果中的第一筆資料。
